## 5. 批量 API 获取输出

命令：`jira-read --live YNOTR-12167 YNOTR-12168 YNOTR-12169`

执行规则：逐个调用 API，单个失败不阻断后续，实时输出进度，完成后汇总。

```
## 批量获取结果（3 个）

| # | Jira ID | 标题 | 优先级 | 状态 |
|---|---------|------|--------|------|

| 2 | YNOTR-12168 | [标题2] | P2 | ✅ 已获取 |
```
