## Contributing

1. Fork → 新增或修改 `skills/<name>/SKILL.md`
2. 运行 `node scripts/gen-skill-docs.mjs` 并提交 `docs/generated/skills-index.md`
3. Pull Request

Skill 编写规范见 [skills/AGENTS.md](skills/AGENTS.md)。仓库目录与协作约定见 [AGENTS.md](AGENTS.md)。
